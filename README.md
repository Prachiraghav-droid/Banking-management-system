# 🏦 Banking Management System

A desktop-based **Banking Management System** developed in **Python** using **Tkinter**. The application provides a simple graphical interface to manage customer accounts, perform banking operations, and store account information using JSON.

---

## 📌 Features

- Create new customer accounts
- Savings and Checking account support
- Deposit money
- Withdraw money
- View account details
- Search customer/account
- Delete accounts
- Automatic account and customer ID generation
- Data stored locally using JSON
- User-friendly GUI built with Tkinter

---

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI)
- Object-Oriented Programming (OOP)
- JSON
- UUID

---


## 🚀 How to Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/banking-management-system.git
```

2. Navigate to the project folder.

```bash
cd banking-management-system
```

3. Run the application.

```bash
python Bank_sys.py
```
